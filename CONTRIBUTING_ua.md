# Як зробити внесок у Cinema Platform Docs

Дякуємо за бажання покращити нашу документацію!

## Філософія проєкту

Мінімалізм і прозорість. Ми цінуємо зрозумілу архітектуру, чистий код і лаконічні пояснення. Уникайте зайвого візуального шуму та надмірних коментарів там, де код або текст говорить сам за себе.

## Кроки для роботи

1. **Гілки:** Використовуйте зрозумілі назви (наприклад, `feat/додано-опис-бд`, `fix/стилі-кнопок`).
2. **Контент:** - Усі статті лежать у `src/content/docs/`.
   - Використовуйте `.md` або `.mdx`.
   - Тримайте тексти короткими, структуруйте інформацію списками та заголовками.
3. **Форматування:** Обов'язково запускайте `bun format` перед створенням коміту. Це автоматично відформатує код і відсортує класи Tailwind.
4. **Pull Requests:** Описуйте суть змін коротко і по суті.

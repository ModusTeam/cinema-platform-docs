# 🎬 Cinema Platform - Documentation

The official documentation for the **Cinema Platform** project. This repository serves as the single source of truth for developers, acting as a central hub that unifies architecture overviews, technical guides, and the REST API specification.

The documentation is built with [Astro Starlight](https://starlight.astro.build/).

## 🗂 Documentation Structure

To ensure a clear developer experience, the materials are divided into three key categories:

- **REST API Reference:** Automatically generated pages based on `swagger.json`. These include detailed descriptions of endpoints, request/response formats, and authorization methods.
- **Architecture:** Overviews of the system design, database structure, service communication, and adopted architectural patterns.
- **Developer Guides:** Step-by-step instructions for local deployment, environment setup, and contributing.

## 🔗 Project Ecosystem

- **Backend (C# / .NET):** [Cinema Platform Backend](https://github.com/ModusTeam/cinema-platform-back)
- **Frontend (React):** [Cinema Platform Frontend](https://github.com/ModusTeam/cinema-platform-front)
- **Live Swagger API:** [MonsterASP Deployment](https://cinematestapi.runasp.net/swagger/index.html)

## 🛠 Contributing Content

- **API Section:** Automatically generated by the `starlight-openapi` plugin based on the remote OpenAPI specification.
- **Articles and Guides:** Add new materials as Markdown (.md or .mdx) files within the `src/content/docs/` directory. Keep the text concise and the structure clear.
